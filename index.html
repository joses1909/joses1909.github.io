ETIQUETAS EN LENGUAJE HTML

HTML:


Esta etiqueta indica el comienzo del documento HTML y también marca el final del mismo.

BODY:


Esta etiqueta envuelve todo el contenido visible del documento HTML, como texto, imágenes, enlaces, etc.

CENTER:


Esta etiqueta se usa para centrar el contenido que está dentro de ella horizontalmente en la página.

H1:


Esta etiqueta define un encabezado de nivel 1, que generalmente se muestra con un tamaño de fuente más grande que el texto normal y a menudo se utiliza para títulos principales.

HR:


Esta etiqueta se utiliza para insertar una línea horizontal en la página, que se puede utilizar para separar secciones de contenido.

BR:


Esta etiqueta se utiliza para insertar un salto de línea, lo que significa que el texto que sigue aparecerá en una nueva línea.

P:


Esta etiqueta define un párrafo de texto.

HEAD:


provee información general (metadatos) acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos.

TITLE:


HTML define el título del documento que se muestra en un browser la barra de título o la pestaña de una página. Solo contiene texto; las etiquetas dentro del elemento se ignoran. Contenido de metadatos. Texto que no es un espacio en blanco entre elementos.
